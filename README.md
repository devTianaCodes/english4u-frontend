# English4U Frontend

React frontend scaffold for the English4U portfolio project.

## Stack
- React
- Vite
- React Router

## Scripts
- `npm install`
- `npm run dev`
- `npm run build`

## Environment
Copy `.env.example` values into a local `.env` file when needed.

## Project Shape
- `src/app`: app bootstrapping and router
- `src/components`: shared layout and UI elements
- `src/features`: feature-specific modules
- `src/pages`: route-level page components
- `src/services`: API utilities
- `src/styles`: reset, theme, and app styles
